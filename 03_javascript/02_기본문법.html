기본 문법

1. 변수

컴퓨터 메모리에 값을 저장하기 위해 할당 받는 공간을 변수라고 한다

1) 변수 선언

변수를 선언할 때는 var, let, const 키워드를 사용한다.

변수의 초기화

변수에 값을 대입(저장)할 때는 대입 연산자(=)를 사용한다.

2. 자료형

자바스크립트 에서는 변수에 자료형을 지정하지 않고 변수에 대입되는 값에 의해 자료형이 결정된다

3. 데이터 형변환

1) 문자열과 숫자의 '+' 연산

2) 형 변환 함수

4. 연산자

'==' 연산자와 '===' 연산자의 차이점

5. 제어문

1) 조건문

2) 반복문
