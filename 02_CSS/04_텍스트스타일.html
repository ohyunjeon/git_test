font-size 속성

텍스트의 크기를 변경할 때 사용한다.

  • px/pt 테스트
  • em/rem 테스트 자식 요소

font-weight

글자의 두께를 변경할 때 사용한다.

  • 원래 굵기
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 가늘게
  • 원래 굵기보다 더 굵게

font-style

텍스트의 기울임을 지정할 때 사용한다.

  • italic 기울임
  • oblique 기울임

font-variant

작은 대문자로 변경할 때 사용한다.

I LOVE YOU / i love you

font

글꼴 스타일을 한번에 지정할 때 사용한다.

  • font 테스트 1
  • font 테스트 2

color

텍스트의 색상을 지정할 떄 사용한다

  • 색상명으로 지정
  • 16진수 값으로 지정
  • rgb/rgba로 지정
  • hsl/hsla로 지정

text-decoration

텍스트에 줄을 긋거나 줄을 없앨 때 사용한다

  • 텍스트 아래에 줄긋기
  • 텍스트 중간에 줄 긋기
  • 텍스트 위에 줄 긋기
  • 하이퍼링크에 밑줄 없애기

text-transform

영문 텍스트의 대소문자 변환시 사용한다

  • none
  • capitalize
  • uppercase
  • LOWERCASE

text=shadow

텍스트에 그림자 효과를 줄 때 사용한다

HTML5

white-space

word-spacing
